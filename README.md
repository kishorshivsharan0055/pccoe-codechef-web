# Codechef PCCOE Chapter

> Made with Next.js

- [Sanity.io](http://sanity.io/)
- [TailiwindCSS](https://typegraphql.com/)
- Formatting with [Prettier](https://prettier.io/)

## Development

### Install Dependencies

```bash
npm install
# or
yarn install
```

### Start Development Server:

```bash
npm dev
# or
yarn dev
```
